Gallery Block allows you to display a set of images using pre-set templates.

 

The Gallery Block displays the contents of an image folder. The layout of the images can be defined using templates.

Choosing images to display

 

 

You can select images to display in the Universal Discovery Widget.

You need to select a folder with images, not a single image, for the Gallery Block to work.

When you select an image folder, the information about the folder will be shown in this window.
