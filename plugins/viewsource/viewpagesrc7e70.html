At any time you can modify an existing Landing Page.

 

After you create a Landing Page, at any time it is possible to change its properties in Edit mode. You can edit a Landing Page through relevant editing options.

If you want to edit eZ Enterprise pages located on a different domain than the page of Admin panel, you need to provide support for CORS (Cross-Origin Resource Sharing) mechanism of your server.

For more information refer to Making cross-origin HTTP requests in the eZ technical documentation.

 

 

Editing a Landing Page

 

 

1. Go to the Page mode.

2. Open a page you would like to edit.

3. Click the Edit button in the Studio toolbar.

An editing page displays relevant options to help you organize content. 

If you start editing a regular Content item (not a Landing Page) from the Page mode, you will be moved to the Content mode.
