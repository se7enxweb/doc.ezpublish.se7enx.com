Are you ready to become a part of the eZ Community? There are several ways in which you can contribute, from spotting and reporting bugs to commenting on the documentation to discussing innovative uses on Slack to coding new bundles.

 

If you're looking to contribute code, whether in form of corrections or separate bundles with features, the open-source nature of eZ Platform lets you do this without any fuss using GitHub. Take a look at GitHub 101 and our Development guidelines to get started.

If you've noticed any improvement needed in the documentation, see how to Contribute to Documentation.

If you'd just like to let us know of a bug you've encountered, see how to report it in Report and follow issues: The bugtracker.

If what you're looking for is simply discussing the way you use eZ Platform, drop by to the community website or the eZ Community Slack team.
