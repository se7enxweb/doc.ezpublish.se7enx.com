Banner Block allows you to insert a banner which opens a link when clicked.

 

A Banner Block is an image Content item with a URL attached to it. The URL opens when the visitor to the Landing Page clicks the image.

Available Banner Block settings

 

 

OptionDescription
Name

Enter the Banner Block name.

ImageUse the Universal Discovery Widget to select an image Content item.
URL

Enter the URL which should open when you click the banner.

 

Banner Block configuration window
