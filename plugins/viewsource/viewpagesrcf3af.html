At any time you can remove existing content from your website.

Removing content does not actually delete it entirely. Instead, removed content is archived and can be restored later when needed.

 

 

 

 

 

 

Removing content in Page mode

 

1. Go to the Page mode.

2. Go to the page you would like to delete.

3. Click Options in the Studio toolbar and select Send to Trash.

 A dialog box appears, asking you to confirm deleting the content.

4. Click Send to Trash.

If you remove a Content item that has children (other content under it in the content tree), both this item and the children will be moved to Trash.

 

 

 

 

Managing Trash

 

You can view Content items in Trash, restore them or delete them permanently from the Content mode. See  Deleting content for more information.
