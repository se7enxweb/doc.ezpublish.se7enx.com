The eZ Publish Mobile Kit

The Mobile Kit provides an overview of how you can do mobile applications development for the eZ Publish Web Content Management System.

The Mobile Kit is a collection of code examples, tutorials, eZ Publish extensions and guidelines that will help you get started with mobile applications development for eZ Publish. It provides all necessary content and knowledge to create functional mobile applications. This includes all kinds of mobile applications such as web, native and hybrid like apps for both IOS and Android platforms.

The eZ Mobile Kit package

The Mobile Kit package content is organized in the following directories: apps, ezpublish, tutorials, white-papers

apps

This folder contains various types of sample mobile applications dedicated to eZ Publish. This includes native, web and hybrid type of apps.

ezpublish

This folder holds eZ Publish extensions required by sample mobile applications included in the Mobile Kit.

tutorials

This folder holds various tutorials which explains how different mobile applications are created. This includes mobile applications for iOS and Android platforms.

white-papers

This folder holds various whitepaper type documents related to the eZ Publish mobile solutions.
